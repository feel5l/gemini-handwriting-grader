Step-by-Step Workflow

Run seven Jupyter notebooks in sequence to go from scanned scripts to final reports.

Generate Answer Sheets

Create personalised, print-ready answer sheets for each student from the name list.

Extract Marking Scheme

AI-parses the marking scheme Word document into a structured, validated Excel workbook.

Define Answer Regions

Intelligent bounding-box detection identifies where students write their answers.

AI-Powered Scoring

Gemini OCR reads handwriting and grades each answer against the marking scheme.

Review & Validate

Interactive web interface lets teachers review, override, and finalise every mark — works offline and on GitHub Pages.

Generate Reports

Modular notebooks produce Excel scores, annotated PDFs, Word and PowerPoint analytics.

Email Distribution

Send personalised score emails with marked scripts and AI performance insights to every student.
